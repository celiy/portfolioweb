Mecarvit

Programa em Python que tem integração com banco de dados PostgreSQL. Este programa permite que o usuário possa registrar clientes e serviços de um modo simples e rápido, dando a opção para ver os clientes e serviços registrados com apenas um clique.

• Registrar cliente
• Regristrar serviço
• Ver clientes
• Ver serviços
• Procurar serviço por nome do cliente
• Mudar serviços para pago ou não pago

Planilhas

Este projeto é um aplicativo em Python que tem o intuido de automatizar o gerenciamento de planilhas do Excel. Além de poder ler os dados de uma planilha, ele também pode inserir dados de forma automatizada. O intuido deste projeto é aprender como manipular e escanear dados de uma planilha.

• Ler planilha
• Inserir dados
• Escanear dados
• Maior/Menor valor
• Média
• Valores repetidos
• Criar planilha
